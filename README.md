# terraform-azure-runbooks

A 30-day hands-on learning journey to become an expert in Terraform and Azure Automation Runbooks (Python).

## Stack
- **Terraform** — Infrastructure as Code for Azure
- **Azure** — Cloud platform
- **GitHub Actions** — CI/CD pipeline (auto-deploy on push)
- **Python** — Azure Automation Runbooks

## Structure
```
terraform/       # Terraform configuration files
runbooks/        # Python automation runbooks
.github/workflows/ # GitHub Actions CI/CD pipelines
```
